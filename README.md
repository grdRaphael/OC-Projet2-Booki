# Booki

Intégration de la page d'accueil de **Booki**, une plateforme fictive permettant de trouver des hébergements et des activités dans la ville de son choix.

Ce projet consiste à intégrer la page d'accueil en **HTML & CSS** à partir des maquettes Figma fournies par l'UI designer, avec un rendu responsive sur trois formats : **desktop, tablette et mobile**.

> Projet 2 — Parcours Développeur Web (OpenClassrooms)


## Lancement en local

Comme le site est statique, aucune installation ni dépendance n'est requise.

### Option 1 — Le plus simple

Téléchargez ou clonez le dépôt, puis ouvrez le fichier `index.html` directement dans votre navigateur (double-clic).

```bash
git clone [URL_DU_DEPOT]
cd booki
```

### Option 2 — Avec un serveur local (recommandé)

Pour un rendu plus fidèle (notamment pour les chemins relatifs), servez le projet via un petit serveur local :

**Avec l'extension Live Server (VS Code)**
Clic droit sur `index.html` → *Open with Live Server*.
